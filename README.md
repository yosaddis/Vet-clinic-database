# 📖 [VET CLINIC] <a name="about-project"></a>

> VET CLINIC is a Postgres project which uses a relational database to create the data structure for a vet clinic.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to:

- Install [Git](https://git-scm.com/)
- Install [Postgres](https://www.postgresql.org/download/)

### Setup

Clone this repository to your desired folder:


```bash
git clone https://github.com/yosaddis/Vet-clinic-database.git
```


### Usage

To use queries provided in this project run psql command and copy all commands one by one in psql shell.
